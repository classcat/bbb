ClassCat® WebConference は クラスキャット が提供する、
SoftLayer クラウド上の Web 会議マネージド・サービスです。

特別な機器は必要ありません。オーディオデバイス付属の PC であれば、ブラウザから手軽に利用できます。Web カメラも利用可能です。

  • 会議室を選択します。
    幾つかの会議室がデフォルトで用意されています。会議室の選択はモデレータの指示に従ってください。特に指示がなければ「会議室（デフォルト）」を選択しましょう。
  • 名前を入力します。
    名前は参加者名になります。識別しやすい名前を入力しましょう。
  • 「参加」ボタンをクリックします。

  • ご利用上の疑問点はモデレータ（会議室の管理者・会議の進行役）にお問合わせください。
  • サービスのご購入につきましては
    下記まで電子メールでお問合わせください。
    クラスキャット・セールスインフォメーション
  • 無料サービスも提供中です。（期間限定サービス）

特徴

レコード & プレイバック

ClassCat® WebConference では後のプレイバックのためにあなたのセッションを録画できるサービスをご用意しています。

ホワイトボード

ホワイトボードであなたのプレゼンテーションの重要な部分に注釈をつけることができます。

デスクトップ共有

あなたのデスクトップをすべてのユーザが視れるようにブロードキャストすることができます。（プレゼンターのみ、最新バージョンの Java が必要です）。

WebRTC オーディオ

Chrome と Firefox ブラウザのユーザは、高品質・低レイテンシの WebRTC オーディオの恩恵を受けることができます。（他のブラウザのユーザは Flash ベースのオーディオをシームレスに利用できます。）

プレゼンテーション

PDF プレゼンテーション資料や MS office の文書をアップロードできます。 ClassCat® WebConference はあなたの現在のスライド、ズーム、パン、注釈、そしてマウスポインタを参加者全員に同期して保持させます。

Web カメラ

複数のユーザで Web カメラを同時に共有できます。同時にアクティブな Web カメラの数にビルトインな制限はありません。
